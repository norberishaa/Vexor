ZeroLogon

CVE-2020-1472

10/10

Patched

2020-08-17

ZeroLogon is a critical vulnerability in Microsoft Active Directory Domain Controllers that allows an attacker to exploit a flaw in the Netlogon protocol. By sending specially crafted messages, an attacker can gain domain admin privileges with almost no effort and no authentication. Because it completely breaks the trust mechanism between domain members and controllers, it became one of the most dangerous Windows vulnerabilities ever disclosed.
